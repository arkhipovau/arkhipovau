## Что взять из каждого

### 1. Mane Tatulyan
**Суть:** академический, текстовый, архивный сайт — не визуальное портфолио, а интеллектуальная экосистема.

**Взять:**
- Табличная структура для CV: Degree / Institution / Thesis / Year / Country
- Разделение на «экосистему» подтем (Books, Journal, Archive) вместо одной плоской страницы
- Длинный narrative bio с сносками — для Research-раздела Julie

**Не брать:** плотность текста на главной — слишком академично для портфолио дизайнера.

---

### 2. Nothing (NL)
**Суть:** продуктовые карточки, tech-эстетика, countdown, glyph.

**Взять:**
- Формула карточки: `название проекта` → `одна острая строка` → `CTA / дата`
- Нумерация и кодовые лейблы (`phone (4a)`) — можно адаптировать как теги дисциплин
- Ощущение «релиза» — для анонса новых проектов

**Не брать:** чисто продуктовую сетку без narrative — Julie нужен контекст, не только тизеры.

---

### 3. Bakken & Bæck (Join)
**Суть:** тёплый скандинавский минимализм, принципы как контент.

**Взять:**
- Блок «Guiding principles»: заголовок-принцип + короткий человеческий абзац
- Ссылка на глубокий culture-раздел (у Julie — About + Research)
- Чистая типографика, generous whitespace

**Не брать:** careers-структуру — не релевантно.

### 4. Claudio Guglieri
**Суть:** feed-first портфолио, local time в хедере, signature cursor.

**Взять:**

- Feed / Work / About / Talks / Press — лёгкая структура разделов
- Один signature interaction (custom cursor с easing) — не больше

**Не брать:** полную feed-модель, если Julie хочет сильный визуальный grid проектов.

### 19. Mark Yareham — Index
**Суть:** монохромный editorial index — весь экран в оранжевом, tone-on-tone типографика, 3-колоночная сетка без картинок (в list view).

**Цвет из CSS** (`style.css`):

**Для Julie (без opacity):**
| Token | Hex | Где |
|-------|-----|-----|
| `--orange` | `#ff7300` | акцентный фон — один блок, не весь сайт |

**Куда на Julie (потом):**
1. **Index** (`/index/`) — оранжевый фон, тёмно-оранжевый текст — контраст с белой главной
2. **Зона крупицы** — один оранжевый eco-блок среди ч/б скролла
3. **Hover / active chapter** — оранжевая pill или underline на `айдентика / сайты / крупицы`

**Не брать:** opacity-систему Mark Yareham — у Julie только solid `#000` / `#fff` + этот оранжевый как третий цвет.

## Топ-7 действий для нового сайта Julie

1. **Открывать с point-of-view**, не с generic tagline — манифест до grid (Shcherban + текущий слоган «celebration, not tournament»).
2. **Нумерованный индекс проектов** с тегами дисциплины + года + статуса (Fernando Espeso + 1/1 Studio).
3. **Писать design rationale** в flagship кейсах — шрифт, концепт, референс (N–E).
4. **Один signature interaction** — local time или hover-zones (Guglieri / Norgram). ~~Custom cursor~~ — отклонено.
5. **Сохранить горизонтальный scroll** как DNA текущего сайта, но на коде с кликабельными кейсами.
6. **Updates-лог** для прессы, новых проектов, вех (Jose Houdini).
7. **Избегать** agency-template (NOHO, GAA): никаких bullet-list services и vague headlines.

### 18. Vanderbrand ⭐ ключевой референс
**Суть:** канадская strategy + creative office, чёрно-белый craft, плотная сетка, accordion-index, live time.

**Типографика из CSS** (`vb.css`):
| Стиль | Desktop | Mobile |
|-------|---------|--------|
| Body / nav / list | **13px / 16px** | 14px / 18px |
| Large text (intro) | **24px**, lh 120% | 20px |
| News body | 13px, lh 130% | — |
| Шрифт | SuisseIntl + RhymesText (news) | — |

**Сетка из CSS:**
- 12 columns, **gutter 4px** (очень плотно)
- margin: 0 → 10px → 20px по breakpoints
- Work grid: **4 колонки**, карточки **4:5**, фон figure `#000`

**Цвет:**
- Только `#000` / `#fff` — без полупрозрачных тонов
- Selection: `#f5f5f5` bg (единственный серый)
- Links: underline `border-bottom: 1px solid` — не цветные ссылки

**Взять:**
- **Solid black/white** — подтверждает твой выбор убрать grey/opacity
- **13/16** как desktop base (близко к твоим 13/15)
- **4px gutter** — ещё плотнее текущих 8px
- Work как **4-col grid** с 4:5 и чёрным фоном под изображением
- **Accordion index** для проектов и news (Index + Updates)
- **Live clock** в footer (`10:20:56` + blinking `•`) — как Guglieri, но внизу
- **mix-blend-mode: difference** на header/nav — белый текст на любом фоне
- Структура кейса: **bold title** + one-line caption под ним
- News/updates как **таблица-аккордеон** (Type / Date)

**Не брать:**
- Text-reveal word animations (пока)
- Video cursor
- Splash screen

**Связь с текущим сайтом Julie:** Vanderbrand ближе всего к тому, куда мы уже идём — чёрный текст, плотная сетка, Arial вместо SuisseIntl, corner nav вместо blend-mode header.

### Паттерн «напряжения»

Сильные референсы держат **низкое межстрочное**:
- 1/1 Studio: **10/12** — самое плотное
- Твой Readymag: **13/15 = 1.15**
- Fernando Espeso: **11.5/16** — мелкий кегль + плотные строки в индексе (`line-height: 15px` в списках)
- Gemini: **lh 1.2** на 12px

**Избегать:** `line-height: 1.45–1.55` на body (как было в v1) — это «воздух агентства», не portfolio craft.

### Для Julie → Arial (3 стиля)

| Стиль | Size | Line-height | Где |
|-------|------|-------------|-----|
| **Text** | 13px | 15px | body, nav, titles, bio, principles |
| **Meta** | 11px | 13px | labels, nums, tags, years, sections, time |
| **Lead** | 14px | 16px | manifesto / h1 только |

### Сетка для Julie

- **12 columns**, margin `8px`, gutter `8px`
- Строки индекса: `padding 4px 0`, без лишних margin между секциями
- Секции: `48–64px` между блоками, не 120px+
